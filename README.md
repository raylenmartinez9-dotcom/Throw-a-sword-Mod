# Sword Throw Mod

A simple Minecraft Forge mod that allows you to throw swords by pressing the R key.

## Features
- Press R to throw the sword in your main hand (if it's a sword)
- The sword will be dropped from your inventory

## Installation
1. Install Minecraft Forge for version 1.20.1
2. Run `./gradlew build` to build the mod
3. Place the generated JAR file in your `.minecraft/mods` folder
4. Launch Minecraft with Forge

## Building
Run `gradlew build` in the project root.

## Note
This is a basic implementation. Throwing swords as projectiles would require more advanced entity creation.
